Racing Robots is a web application I built based on the puzzle board game "Ricochet Robots", designed by Alex Randolph and published by Rio Grande Games.

It is available here: https://rr-421.web.app/

The objective of the game is to reach the target with the robot of the same color in as few moves as possible.
